### Tricks
- append string to file
```sh
echo 'source "/etc/profile.d/rvm.sh"' >> ~/.zshrc

```

### Install Rails
```sh
gem install rails -v "~>5.2"
```

### Uninstall Rails Completely
```sh
gem uninstall rails
gem uninstall railties
```
